# python-algorithm-lab
A dictionary is a data structure that stores information in pairs: a key and a value.
Each key is unique, and it is used to access its corresponding value.
think of it as:
The word is the key.
The definition is the value.

Linear Search?
Linear Search is a method of finding an element in a list by checking each item one by one from the beginning until the target is found or the list end

Binary Search?
Binary Search is a method of finding an element in a sorted list by repeatedly dividing the search range in half.
